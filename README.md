# OpenClaw `ghst` Skill

This repository contains the `ghst` skill for OpenClaw, allowing your AI agent to interact directly with any Ghost blog or publication via the Ghost Admin API. 

It acts as a wrapper around the [@tryghost/ghst](https://github.com/TryGhost/ghst) CLI tool.

## Usage

Once installed and configured, your OpenClaw agent can manage posts, pages, members, tags, newsletters, themes, and more on your Ghost instance.

For full details on agent capabilities, command references, and specific workflows, please see the primary [SKILL.md](SKILL.md) instruction file.
